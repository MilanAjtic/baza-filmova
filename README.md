# Baza filmova

Baza filmova, pokazna React aplikacija Škole koda.

Podatke vuče iz [baza-filmova-api](https://github.com/skolakoda/baza-filmova-api).

## Pokretanje

Prvo kloniraj repo, pa:

```
npm i
npm start
```
